# Screenshots

This directory contains application screenshots for the README.

## Files:
- `1.png` - Home page (dark mode)
- `2.png` - Portfolio page
- `3.png` - Apps page  
- `4.png` - Home page (light mode)

## How to Update Screenshots:

1. Take screenshots of each page at a reasonable resolution (1920x1080 or similar)
2. Name them according to the convention above
3. Replace the existing files in this directory
4. Screenshots will automatically appear in the main README.md

## Tips:
- Use PNG format for better quality
- Keep file sizes reasonable (< 500KB per image)
- Ensure screenshots show the best features of the application
- Update screenshots when making significant UI changes
